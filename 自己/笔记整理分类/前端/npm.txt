
npm初始化 //创建package.json
	使用命令 npm init  填写相关信息 或是使用  npm init --yes  直接使用默认的配置类创建
	npm init --yes 简写  npm init -y



设置淘宝镜像
	npm install -g cnpm --registry=https://registry.npm.taobao.org
	注意：很多模块找不到等问题都是npm命令的问题，这是使用cnpm可以解决很多莫名其妙的问题。


npm重新注册
	npm config set registry "http://registry.npmjs.org/"             //这个就是npm代理设置错误，重新设置代理
